# Helix Calendar

A recursive 3D time-helix calendar where nested coils represent time from centuries down to seconds.

## Development

```bash
npm install
npm run dev
```

Opens at http://localhost:3000

## Controls

- **Left-click drag** — Orbit / rotate
- **Right-click drag** — Scrub through time
- **Scroll wheel** — Zoom between time levels
- **Click time display** — Jump to specific date
- **"+" button** — Create event
- **"● Now" button** — Return to current time

## Tech Stack

React, Three.js (React Three Fiber), Zustand, date-fns, Dexie, Vite, TypeScript
